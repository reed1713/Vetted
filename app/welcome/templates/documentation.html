scrape notes

Status

Edit Detection Type form

priority: high, medium, low

status: open, reviewing, vetted, stale

tags: seperated by , . leading and trailing whitespaces will be removed from each entry

notes: notes and stuff

indicators:

BRO_INTEL
indicator type : indicator value. indicators are split by r\n. all whitespace will be removed, as well as any \r before inputting into to the database. reference the Bro_Intel type for the expected indicator type values.
SNORT
signatures seperate by a space or \r\n\r\n>. remove leading a trailing whitespace to avoid errors.
YARA
signatures seperate by a space or \r\n}\r\n\r\n>. remove leading a trailing whitespace to avoid errors.

Lists

whitelist

any values in this list will be excluded from the bro intel scrape. uses python regex when it runs against the match list. scrapes will fail if there are any carriage return newline feeds, or whitespaces in this file.

keywords

any values in this list will be used to create a list of tags. uses python regex when it runs against the match list. scrapes will fail if there are any carriage return newline feeds, or whitespaces in this file.

API

GET /api/vetted/network_bro_intel/json/ - pull all vetted bro intel json objects
GET /api/vetted/network_bro_intel/json/{id} - pull a vetted bro intel json object
GET /api/vetted/network_snort/json/ - pull all vetted snort json objects
GET /api/vetted/network_snort/json/{id} - pull a vetted snort json object
GET /api/vetted/binary_yara/json/ - pull all vetted yara json objects
GET /api/vetted/binary_yara/json/{id} - pull a vetted yara json object
